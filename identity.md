## Positioning

| | |
|---|---|
| **Genre** | Action-Rogue-lite / Isometric 3D Hack 'n' Slash |
| **Tone** | Cynical, reality-TV satire, B-movie gore + prime-time confetti |
| **Audience** | Action roguelite fans (*Hades*, *Diablo*) and streaming culture |
| **Differentiation** | Spectacle before survival · 4th wall (chat = audience) · sadistic Showrunner |

**Inspirations** (list at the **bottom**, never as the main hook):  
*Smash TV*, *The Running Man*, *Hades*, *Dungeon Crawler Carl* (world, cynicism).

> Do not position the game as "Dungeon Crawler Carl as a roguelite."  
> Own identity = **deadly reality TV + audience ratings + "Actor" Agency**.

## Canonical vocabulary

| Official term | Definition | ❌ Avoid |
|---------------|------------|----------|
| **"Actor" Agency** | Structure the player runs; recruits and sends candidates (quotes = TV irony) | Gladiator company |
| **Disposable person** | Individual recruited and sent into the dungeon; death is definitive at run end | Disposable clone, clone |
| **Contestant** / **Participant** | In-run / reality-TV wording for the controlled person | Gladiator (for the player) |
| **Agent** / **Casting Director** | The player as manager (flavor, optional) | — |
| **Agency Office** | Meta-progression hub (recruitment, lobbying, upgrades) | Company hub, corporation |
| **Showrunner** | Sadistic AI running the broadcast dungeon; talks to chat, not the contestant | — |
| **Hype** | Audience gauge influencing economy and events | — |

### Diegetic UI (in-game)

- "Status: **Disposable**"
- **Disposable™** wristband
- "**Casting** number: #XXXX" (not "clone number")

### Legitimate exceptions

- **`antiquity` theme**: NPC "series gladiator" — in-universe décor, not the player.
- **Lighting**: "stadium set" / "extras under spotlights" — visual metaphor, not the pitch.
