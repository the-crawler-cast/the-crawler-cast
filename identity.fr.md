## Positionnement

| | |
|---|---|
| **Genre** | Action-Rogue-lite / Hack 'n' Slash isométrique 3D |
| **Ton** | Cynique, satire télé-réalité, gore série B + confettis prime-time |
| **Cible** | Fans de roguelites action (*Hades*) et culture streaming |
| **Différenciation** | Le spectacle avant la survie · 4ème mur (chat = public) · Showrunner sadique |

**Inspirations** (citer en **bas de liste**, jamais en accroche principale) :  
*Smash TV*, *The Running Man*, *Hades*, *Dungeon Crawler Carl* (univers, cynisme).

> Ne pas positionner le jeu comme « le Dungeon Crawler Carl du roguelite ».  
> L'identité propre = **télé-réalité mortelle + audimat + agence d'« acteurs »**.

## Vocabulaire canonique

| Terme officiel | Définition | ❌ Éviter |
|----------------|------------|----------|
| **Agence d'« Acteurs »** | Structure que dirige le joueur ; recrute et envoie des candidats (guillemets = ironie TV) | Société de gladiateurs |
| **Personne jetable** | Individu recruté et envoyé dans le donjon ; mort définitive à la fin de la run | Clone jetable, clone |
| **Candidat** / **Participant** | Terme in-run / langage télé-réalité pour la personne contrôlée | Gladiateur (pour le joueur) |
| **Agent** / **Directeur de casting** | Le joueur en tant que manager (flavor, optionnel) | — |
| **Bureau de l'agence** | Hub méta-progression (recrutement, lobbying, upgrades) | Hub société, corporation |
| **Showrunner** | IA sadique animant le donjon médiatisé ; parle au chat, pas au candidat | — |
| **Hype** | Jauge d'audimat influençant économie et événements | — |

### UI diégétique (in-game)

- « Statut : **Jetable** »
- Bracelet **Jetable™**
- « Numéro de **casting** : #XXXX » (pas « numéro clone »)

### Exceptions légitimes

- **Thème `antiquity`** : PNJ « gladiateur de série » — décor in-universe, pas le joueur.
- **Éclairage** : « plateau de stade » / « figurants sous projecteurs » — métaphore visuelle, pas le pitch.
